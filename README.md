# RC 골조 적산 프로그램 (프로토타입)

철근 콘크리트(RC) 구조물(기둥, 보, 슬래브, 벽체, 기초)의 콘크리트 부피(㎥), 거푸집 면적(㎡), 철근 중량(kg/ton)을 신속하게 산출할 수 있는 프로토타입 프로그램입니다.

## 📂 파일 구성
- **[rc-적산-프로토타입.html](rc-적산-프로토타입.html)**: 
  - 외부 빌드 도구 없이 브라우저에서 바로 실행 가능한 **HTML 단일 파일 버전**입니다.
  - 다크 테마(Slate) 디자인, 부재 수동 입력 및 산출, Excel 다운로드(SheetJS 탑재), DXF 레이어 매핑 등의 기능이 내장되어 있습니다.
- **[rc-적산-프로토타입.jsx](rc-적산-프로토타입.jsx)**: 
  - React 프로젝트 환경에 통합할 수 있는 **React 컴포넌트 버전**의 코드 파일입니다.

## 🚀 실행 방법 (HTML 버전)
1. `rc-적산-프로토타입.html` 파일을 더블 클릭하여 Chrome, Edge 등의 웹 브라우저로 실행합니다.
2. 부재의 치수 및 사양을 입력하면 자동으로 산출근거와 수량이 계산됩니다.
3. 계산된 집계 내역은 **Excel 다운로드** 버튼을 눌러 `.xlsx` 파일로 내보낼 수 있습니다.

## 📐 물량 산출 기준 (간략식)
이 프로그램은 기본 구조 계산용 간략식을 활용하고 있으므로, 실무 적용 전에는 반드시 정밀 검증이 필요합니다.
- **공통**: 정착 및 이음길이, 피복 두께 공제, 개구부 면적 공제는 계산에서 제외되어 있습니다.
- **기둥**: 콘크리트는 단면적 $\times$ 높이, 거푸집은 사면 둘레 $\times$ 높이로 산출합니다.
- **보**: 콘크리트는 단면적 $\times$ 길이, 거푸집은 밑면 및 양 측면만 고려하여 산출합니다.
- **슬래브**: 콘크리트는 면적 $\times$ 두께, 거푸집은 하부 면적만 산출합니다.
- **벽체**: 콘크리트는 면적 $\times$ 두께, 거푸집은 양면(2배)으로 산출합니다.
- **기초**: 콘크리트는 면적 $\times$ 두께, 거푸집은 측면 면적으로 산출합니다.
